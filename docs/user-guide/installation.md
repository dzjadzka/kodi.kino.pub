# Installation

Install the kino.pub add-on either from the Kodi repository ZIP (for automatic updates) or manually from a release ZIP.

## Prerequisites
- Kodi 19+ for add-on versions 3.x.x; for Kodi 17/18 use version 2.12.4.
- Network access to download ZIP files.

## Option A: Install from repository ZIP (recommended for updates)
1. Download the repository ZIP: <https://kodi-kino-pub.netlify.app/repo.kino.pub.zip>.
2. In Kodi, open **Add-on manager**.
3. Choose **Install from ZIP file** and select the downloaded repository ZIP.
4. Return to **Add-on manager** → **Install from repository**.
5. Open **Kino.pub Add-on Repository** → **Video add-ons** → **kino.pub** and install.

## Option B: Manual installation (single release ZIP)
1. Download the latest release ZIP from <https://github.com/quarckster/kodi.kino.pub/releases/latest>.
2. In Kodi, open **Add-on manager**.
3. Choose **Install from ZIP file** and select the downloaded release ZIP.

## Verify
- After installation, the add-on should appear under **Video add-ons**. Launch it to confirm it opens without errors.
