# kino.pub Kodi add-on overview

## Purpose
- Kodi video add-on for kino.pub online movie theater (streams movies/series via the kino.pub service).
- Provides video content via Kodi plugin source (`provides=video`).

## Compatibility
- Versions **3.x.x** work with **Kodi 19+**.
- For **Kodi 17/18**, use **2.12.4** releases.

## Key features (from README)
- Choose video quality before playback; supports 4K.
- Configurable start screen items.
- Custom context menu items: "Will watch", "Mark as seen", "Change bookmarks", "kino.pub comments".
- Watch/resume points synced across devices; includes "Hot" and "Watching" folders.
- Trakt.tv support.
- Supports [inputstream.adaptive](https://github.com/peak3d/inputstream.adaptive) add-on.
- Search history.
- HTTP and SOCKS proxy support.
- Localizations: en_GB, ru_RU, uk_UA.
